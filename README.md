# Sales Data Analysis Project

## Overview
This project is based on sales data analysis using Microsoft Excel.

## Tasks Performed
- Data Cleaning
- Monthly Sales Trend Analysis
- Product Analysis
- Payment Method Analysis
- Order Status Analysis
- Referral Source Analysis
- Basic Statistics

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- SUM()
- AVERAGE()
- MIN()
- MAX()
- COUNT()
- IF()
